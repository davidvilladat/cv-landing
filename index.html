About

Crew intelligence,
built by crew.

I'm a Cristian Villada, Etihad Airways crew based in Abu Dhabi. These tools started as personal spreadsheets for ZED travel planning and grew into full interactive applications used across the fleet.

Each project draws from real data sources — Cirium schedules, FlightRadar24 live feeds, and the ZED/LT fare matrices — and is designed to surface insights that matter on layovers, during bidding, or before booking a jump.

Everything is for crew, by crew. Free, no login, works on any device.
